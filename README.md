## ✨ About

**Mathodic** is a deterministic **math-to-music engine** where every character in your equation becomes part of a musical composition.

Digits become notes, operators change rhythm, functions create arpeggios, and the `=` sign finally resolves everything into a satisfying chord.

**Same equation → Same melody.**

Change one character and the song changes too.

> No AI generated your melody.
>
> It's just math showing off.

---

## 🚀 Features

| Feature | Description |
|---------|-------------|
| 🎼 Deterministic Engine | Same input always creates the exact same melody. No randomness. No excuses. |
| 🧮 Linear Equation Solver | Solves equations like `3x+5x=10 → x=1.25` faster than searching YouTube tutorials. |
| 🎹 Layered Synth Engine | Multiple synth layers, filters, reverb, stereo spread and soft saturation for richer sound. |
| 🌊 3D Signal Wave | Beautiful real-time waveform visualization that dances with every note. |
| 🦋 Butterfly | Nobody asked for it. Everyone likes it anyway. |
